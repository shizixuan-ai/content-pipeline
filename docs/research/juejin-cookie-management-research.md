**Features**:

## Decision

**Verdict**: Hybrid

**Recommended candidate**: agent-browser + 辅助包装脚本

**Rationale**:
agent-browser 已安装在 toolchain 中，提供了完整的 browser state 管理能力。需要自建的是：
1. Cookie 格式化脚本（agent-browser cookies get 的 JSON → Cookie header 字符串）
2. 扫码登录引导脚本
3. 与现有 pipeline 的集成（post-juejin.sh 调用前自动获取 cookie）

**Integration scope**:
- 新建 `scripts/login-juejin.sh` — 扫码登录引导 + 状态持久化
- 新建 `scripts/get-juejin-cookie.sh` — 从 session 提取并格式化 Cookie header
- 新建 `scripts/check-juejin-cookie.sh` — 检测 + 自动续期包装器
- 修改 `post-juejin.sh` — 集成自动 cookie 获取逻辑
- 修改 `auth-juejin.sh` — 支持从 session 检测（不仅仅是 env var）
