# 文章要点

## 基本信息
- **标题（暂定）**: 聚焦内容，剩下的交给 Pipeline — 我的全自动多平台分发工具
- **文章类型**: 产品发布类
- **目标受众**: 技术博主、独立开发者、技术写作者

## 核心要素
- **痛点**: 写文章容易，分发到多平台太麻烦。每个平台格式/语气/受众不同，手动适配耗时费力
- **解决方案**: content-pipeline — 一次写作 → 按平台人格化改写 → API 一键分发。6 平台 5 分钟完成
- **效果量化**: 6 个平台（掘金、公众号、头条、知乎、简书、CSDN），5 分钟一键分发
- **金句**: 聚焦内容，剩下的交给 pipeline
